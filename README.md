# bash
バッシュ関連のファイル<BR>
カレントディレクトリでgrep出来るようにカスタマイズしています<BR>
Bash related files.<BR>
It is customized to be able to grep in the current directory.<BR>

### .bashrc
bashのログイン時に毎回読み込まれるファイル<BR>
A file is read every time you login to bash.<BR>

### .bash_profile
ユーザー毎に読み込まれるファイル<BR>
File loaded per user.<BR>

### .bash_logout
bashのログアウト時に毎回読み込まれるファイル<BR>
A file is read every time you log out of bash.<BR>

### .bash_login
bashのログイン時に~/.bash_profileが存在しなければ読み込まれるファイル<BR>
The file is read if ~/.bash_profile does not exist when you login to bash.<BR>
