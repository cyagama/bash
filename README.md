# bash
バッシュ関連のファイル
カレントディレクトリでgrep出来るようにカスタマイズしています
Bash related files.
It is customized to be able to grep in the current directory.

### .bashrc
bashのログイン時に毎回読み込まれるファイル
A file is read every time you login to bash.

### .bash_profile
ユーザー毎に読み込まれるファイル
File loaded per user.

### .bash_logout
bashのログアウト時に毎回読み込まれるファイル
A file is read every time you log out of bash.

### .bash_login
bashのログイン時に~/.bash_profileが存在しなければ読み込まれるファイル
The file is read if ~/.bash_profile does not exist when you login to bash.
